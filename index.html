Модуль: password_generator

class PasswordGenerator()

Класс для генерации случайных безопасных паролей. Используется на backend-сервере для предоставления утилиты "Генератор паролей" конечным пользователям.

generate(length: int = 12, use_special_chars: bool = True) → str

Генерирует случайный пароль заданной длины, используя криптографически стойкий модуль random.

ПараметрТипОписание
lengthintДлина генерируемого пароля (по умолчанию 12).
use_special_charsboolФлаг использования спецсимволов (!@#$%^&*).
ВозвращаетstrСгенерированная строка пароля.
ИсключенияValueErrorЕсли длина пароля меньше 4.
